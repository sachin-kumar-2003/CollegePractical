CSS Usage Examples

1. Inline CSS

This paragraph demonstrates inline CSS. The style attribute is used directly on the HTML element.

<p style="color: blue; font-weight: bold;">This paragraph demonstrates inline CSS.</p>

2. Internal CSS

This paragraph is styled using internal CSS defined in the <style> tag in the <head> section.

<style>
    p {
        color: gray;
        font-size: 16px;
    }
</style>
